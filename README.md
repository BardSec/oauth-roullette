# oauth-roulette

A collection of OAuth-powered web apps that do something fun with your account data.

## Projects

### [email-roulette](email-roulette/)

Sign in with your Microsoft 365 account, spin the wheel, and reveal a random email from your junk folder.

- **Auth:** MSAL Node (Authorization Code flow)
- **Backend:** Node.js + Express
- **Graph API:** Microsoft Graph `Mail.Read` / `User.Read`
- **Frontend:** Vanilla HTML / CSS / Canvas

See [email-roulette/README.md](email-roulette/README.md) for full setup and deployment instructions.
